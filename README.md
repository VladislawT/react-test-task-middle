## 📋 О проекте

Интернет-магазин, реализованный в рамках тестового задания. Включает каталог товаров, детальную страницу товара и корзину покупок. Данные о товарах получаются из псевдо-API.

### Основной функционал

-  **Поиск товаров** - поиск по названию
-  **Фильтрация по категориям** - выбор категории товаров
-  **Сортировка** - по цене (возрастание/убывание)
-  **Корзина покупок** - добавление/удаление товаров, изменение количества
-  **Сохранение состояния** - данные корзины сохраняются в localStorage
-  **Адаптивный дизайн** - корректное отображение на всех устройствах

## Технологии

- **Framework:** Next.js 15 (App Router)
- **Язык:** TypeScript
- **Стилизация:** TailwindCSS
- **State Management:** Zustand
- **Иконки:** React Icons
- **Деплой:** Vercel
